# Projeto Zelda Mini Clone

## Objetivo do Projeto
O projeto foi desenvolvido como o trabalho final da disciplina optativa "Desenvolvimento de Jogos", do meu curso atual de Bacharelado em Sistemas de Informação no 7° semestre.
A aplicação visa provar minhas competências no desenvolvimento de um jogo simples, seguindo as instruções passadas pelo professor no decorrer das aulas.

O jogo desenvolvido foi *inspirado* e utiliza assets do jogo The Legend of Zelda do NES, tomando liberdades criativas na arte e nas mecânicas, resultando em um jogo curto que demonstra minhas abilidades desenvolvidas estudando a matéria. 
O game conta com 4 fases, com 3 inimigos distintos, uma arma, sistema de ataque, HUD, sistema de inventário, vida, transição de cenário e pontuação.

## Como executar
Primeiramente o projeto deve ser clonado através do git. Logo após basta acessar a arquivo no caminho 'JavaMiniClone/Zeldaminiclone/ZeldaMiniClone.jar'.
